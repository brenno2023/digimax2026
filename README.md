# Sistema Web Projeto Digimax

## Sobre o Projeto

O Sistema Web Projeto Digimax foi desenvolvido como parte do Projeto de Extensão Comunitária da Fatec São José do Rio Preto.

O sistema tem como objetivo auxiliar a gestão de uma gráfica, permitindo o cadastro de clientes, gerenciamento de pedidos, solicitação de orçamentos, acompanhamento de serviços e administração interna da empresa.

* Solicitação de orçamento

* Consulta de pedidos
* Resposta de orçamentos

### Administrador

* Cadastro de funcionários
* Gerenciamento de clientes
* Gerenciamento de funcionários

## Banco de Dados

O banco de dados foi desenvolvido em SQL Server e contém:

* Tabelas relacionais
* View
* Stored Procedure
* Function
* Trigger
* Inserts para testes e validação

## Como Executar

1. Executar o script SQL do banco de dados.
2. Configurar a conexão com o SQL Server.
3. Executar a aplicação Spring Boot.
4. Acessar o sistema pelo navegador.
